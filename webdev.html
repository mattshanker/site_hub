Responsive Web Design is the technique of using media queries, adaptive content and grid layout to allow your site to respond to fit any screen. Things like adding a schedule, color picker, menu, a date picker help make your site more useful to first-time, repeat, and potential customers.

The important thing is to not let the tool of responsive design become the goal. The real goal is to build a website that is both fast and useful, for everyone.

As of April 2015, Google's search algorithm has already started penalizing websites that aren't optimized for mobile in results, so it's more important than ever to design with mobile in mind. [TechCrunch]
