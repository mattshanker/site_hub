Dynamic vs Static

How does a site get it's content?

Static sites use only HTML, CSS, and JavaScript. Updates to the site come from making changes to these files, which are also called a website's source code. Dyanmic sites, which run server code such as PHP in addition to HTML, CSS and JavaScript, receive content updates via a CMS like WordPress. For anyone who doesn't need to change their content more than 1 or 2 times a month and would rather have a more secure and maintainable site, a static site is the best choice.

Landing pages - your site's welcome mat

What happens after someone clicks on your ad? Or your email blast?
